# ScamBridge

## Overview
ScamBridge is a prototype tool designed to help users identify and respond to potential investment scams, especially for individuals with limited English proficiency.

## Problem
Many people fall victim to financial scams due to language barriers, unclear messaging, and lack of accessible tools to evaluate suspicious information.

## Solution
ScamBridge allows users to input or upload investment-related messages and analyzes them for potential fraud signals, returning simplified and translated insights.

## Features
- Multilingual support for accessibility
- AI-assisted scam detection
- User-friendly interface for evaluating suspicious messages
- Focus on financial and investment-related fraud

## Goal
To make financial safety tools more accessible and help users make informed decisions when evaluating potential scams.

## Status
Early-stage prototype. Actively being developed and improved.
